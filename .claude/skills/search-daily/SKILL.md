# 每日媒体报道搜索

## 核心原则
- **只搜索、只报告，不自动归档** — 发现新报道后列出来给用户确认
- **全量搜索** — 不限时间范围，新报道和历史遗漏都要找
- 目标：找到 100 篇真实有效的报道链接（当前进度见 goals.md）

## 操作流程

### 1. 加载策略和现状

读取 `search-strategy.md`（如果存在）获取当前搜索策略。
读取 `goals.md` 获取当前进度。
读取 `sources/TopicReportList.MD` 提取所有已收录 URL（用于去重）。

### 2. 执行搜索

**按策略搜索**（如果 search-strategy.md 存在，优先用里面的关键词和方法）

**默认关键词**：

第一层（人名直搜）：
- "廖兰新"

第二层（人名+公司）：
- "廖兰新 滴滴"
- "廖兰新 小桔充电"
- "廖兰新 滴滴能源"
- "廖兰新 小桔能源"
- "廖兰新 滴滴充电"
- "廖兰新 滴滴电力"

第三层（职位）：
- "滴滴能源 CTO"
- "小桔充电 CTO"
- "滴滴充电 CTO"

第四层（产品/事件，轮换使用）：
- "Orange Unicorn 充电桩"
- "独角兽计划 充电"
- "滴滴 智能运维 充电"
- "滴滴 车网互动"
- "滴滴 超充"
- "小桔充电 智能运维"

第五层（行业大会，轮换使用）：
- "中国汽车充换电生态大会 滴滴"
- "中国汽车论坛 滴滴充电"
- "充换电技术创新大会 滴滴"

**搜索节奏**：
- 不着急，控制 API 调用频率
- 每天只搜 1-2 层关键词，轮换覆盖
- 一周内覆盖所有层级即可
- 宁可慢一点，不要频繁调用接口

**搜索工具**：
1. **Tavily Search** — 通用网页搜索（主力）
2. **weixin-search** — 微信公众号+腾讯体系搜索（关键补充）
   - 使用 `/weixin-search 廖兰新` 搜索微信公众号文章
   - 搜狗微信索引是微信文章唯一可靠来源
   - 深度模式：`/weixin-search --deep 廖兰新 滴滴充电`

**全量搜索策略**：
- 不限定时间范围，每次用不同的搜索角度
- 轮换策略：
  - 第 1 天：通用搜索（Tavily 各关键词全量）
  - 第 2 天：微信公众号搜索（weixin-search 全部关键词）
  - 第 3 天：高产来源定向搜索（site:chd.in-en.com 等）
  - 第 4 天：行业大会/事件关联搜索（搜大会名+演讲嘉宾）
  - 第 5 天：转载链追踪（从已知报道找同源转载）
  - 第 6 天：长尾关键词搜索（"滴滴充电负责人"、"小桔能源"等变体）
  - 第 7 天：新渠道探索 + 微信深度调研（weixin-search --deep）
- 以上为默认轮换，如果 search-strategy.md 有更好的安排则优先

**每个关键词都要记录效果**：搜到几条、新发现几条、来自什么来源。

### 3. 去重

- URL 精确匹配已有记录 → 排除
- 内容明显不相关 → 排除
- 同一篇文章不同 URL（转载） → 都算有效，都保留

### 4. 写搜索日志

追加一行到 `search-log.jsonl`：
```json
{

  "search_angle": "通用搜索/定向搜索/转载追踪/...",

    "廖兰新 滴滴能源": {"total": 3, "new": 0, "duplicates": 3}
  },
  "source_hits": {"chd.in-en.com": 2, "ifeng.com": 1},
  "new_reports": [
    {"url": "...", "source": "来源名", "title": "标题", "keyword_found_by": "廖兰新"}

  "notes": "观察记录"
```

**有新发现：**
```

发现 X 篇新报道，请确认是否归档：

1. [来源] 标题
   🔗 URL

2. ...

回复"确认"我来归档，或告诉我哪些不要。

📊 进度：XX / 100（XX%）
```

**无新发现：**
```
📰 报道监测日报 [日期]
今日搜索角度：[xxx]
未发现新报道。
📊 进度：XX / 100（XX%）
```
