# 廖兰新 — 公开演讲与媒体报道汇总

滴滴能源 CTO 廖兰新的公开演讲、行业发言与媒体报道时间线页面（2020–2025）。

## 概览

本项目是一个纯静态 HTML 页面，以时间线（Timeline）形式记录了廖兰新在新能源充电行业的公开活动轨迹，包括：

| 类别 | 数量 |
|------|------|
| 公开发言 | 13 次 |
| 专利申请 | 347 件 |
| 已授权专利 | 121 件 |
| 媒体报道来源 | 60+ 家 |

## 关键数据亮点

- **知识产权**：347 件国内外专利申请，121 件已获授权，覆盖智能充电全链路
- **车网互动**：需求响应 ~4,000 小时，调峰电量 >1,000 万千瓦时，响应精度 95%+，累计减碳 5,500+ 吨，接入 40+ 城市电力市场
- **智能运维**：引入 150+ 运维服务商，设备可用率从 93.5% 提升至 98.7%，累计处理超 1 万笔维修工单，约 20 万设备完成智能化升级
- **行业标准**：参编《电动汽车充电设施智能运维技术白皮书》（中电联）、《2024 中国充电基础设施服务质量发展报告》
- **业务覆盖**：滴滴充电（原小桔充电）服务 270+ 城市、62,000+ 座充电站、在线快充枪超 26 万把、累计合作商户 11,000+、碳减排约 1,700 万吨

## 页面特性

- 纯静态 HTML，无需构建工具，浏览器直接打开即可
- 响应式设计，适配桌面与移动端
- 暗色主题，琥珀/金色点缀的 "Electric Archive" 视觉风格
- 使用 Outfit + Noto Sans SC + JetBrains Mono 字体组合
- glassmorphism 卡片、噪点纹理、滚动触发动画
- 所有报道均附原文链接，可溯源验证
